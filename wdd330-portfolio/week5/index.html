I feel like most people thing that debugging is pretty easy, but there is a lot that goes into if you want to do it well.
You can use console.log() to display JavaScript in the debugger window.
You can use the keyword debugger to stop JavaScript execution and call the debugging function.
It is similar to a breakpoint.
Example: (From W3schools)
) let x = 15 * 5;
debugger;
document.getElementById("demo").innerHTML = x;
