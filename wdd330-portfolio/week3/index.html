Notes for week 3:

Something that I thought was interesting was the This keyword.
The This keyword allows us to acces an object.
Here is an example of the This keyword in action.
let user = {
name: "John",
age: 30,

sayHi() {
alert(this.name);
}

};

user.sayHi(); // John
In this example the value of This will be "user".
The value of This is evaluated during the run-time, depending on the context.
