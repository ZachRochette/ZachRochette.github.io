This week was really good for me because I need to put some more practice into JavaScript. So here are a couple of things that helped me out and that I need to REMEMBER.
1. Truthy and Falsy values - "Truthy values are expressions that evaluate to the boolean value of true, and falsy values are expressions that evaluate to the boolean value of false."
Falsy expressions - "False, NaN, Undefined, Null, Empty String (""/"), and 0"
Truthy expressions - Anything else other than those six expressions
2.Template Literals - "Template literals are literals delimited with backticks (`), allowing embedded expressions called substitutions."
Pretty much template literals allow you to print strings more efficiently, which is nice.

3. A couple of properties to keep in mind

  • .length
  • .toUpperCase()/toLowerCase()
  • .charAt(i)
  • .includes(i)
  • .indexOf(i)
  • .endsWith()
  • .startsWith()

4. Recursion - "Recursion is when a function calls itself until someone stops it. If no one stops it then it'll recurse (call itself) forever."
Example of Recursion -
function power(base, exponent) {
if (exponent == 0) {
return 1;
} else {
return base * power(base, exponent - 1);
}
}
